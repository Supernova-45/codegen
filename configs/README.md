# Configs

Kept config files:

- `mvp_mbpp.yaml`: default MBPP run preset.
- `mvp_humaneval.yaml`: default HumanEval run preset (supports optional `humanevalplus` re-scoring block).
- `mvp_humaneval_google_oneshot.yaml`: one-shot preset used by the Google wrapper script.
- `model_profiles.yaml`: profile mappings for `scripts/run_model_matrix.py`.
